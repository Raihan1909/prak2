1. Mengapa Menggunakan HTML, CSS, dan JavaScript?

1. HTML (HyperText Markup Language)

HTML digunakan untuk mendefinisikan struktur dasar sebuah halaman web. Elemen-elemen seperti paragraf, heading, tabel, gambar, dan tautan semuanya diatur menggunakan HTML. HTML adalah bahasa markup yang wajib digunakan karena berfungsi sebagai pondasi untuk konten yang akan ditampilkan.

2. CSS (Cascading Style Sheets)

CSS digunakan untuk mengatur tampilan dan tata letak halaman web. Dengan CSS, pengembang dapat menentukan warna, ukuran font, margin, padding, dan bahkan tata letak kompleks seperti grid atau flexbox. Tanpa CSS, halaman web hanya akan tampil dengan elemen-elemen dasar yang terlihat kurang menarik.

3. JavaScript

JavaScript adalah bahasa pemrograman yang memberikan kemampuan interaktif pada halaman web. Dengan JavaScript, halaman dapat merespons tindakan pengguna, seperti klik, gulir, atau masukan data. JavaScript juga memungkinkan pengembangan fitur yang kompleks seperti animasi, permainan, atau aplikasi web berbasis data.

2. Bagaimana HTML, CSS, dan JavaScript Diintegrasikan?

1. HTML Sebagai Struktur Utama

File HTML adalah inti dari sebuah halaman web. Elemen-elemen di dalamnya disusun dalam bentuk hierarki.

2. CSS untuk Desain

CSS dapat dihubungkan ke file HTML dengan tiga cara:

  • Inline: CSS langsung ditulis di atribut elemen HTML.
  • Internal: CSS ditulis di dalam tag <style> di bagian <head>.
  • Eksternal: CSS diletakkan dalam file terpisah (misalnya styles.css) dan dihubungkan menggunakan tag <link>.

3. JavaScript untuk Interaktivitas

JavaScript dapat dimasukkan ke dalam halaman web melalui cara berikut:

  • Inline: Ditulis langsung pada elemen HTML menggunakan atribut onclick, onchange, dll.
  • Internal: Ditulis di dalam tag <script> pada file HTML.
  • Eksternal: JavaScript diletakkan di file .js yang dihubungkan dengan tag <script>.
